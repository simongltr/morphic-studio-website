# Iteration log

One line per iteration: what changed and why. Newest at the top.

- **v2** — 2026-05-31 — type and palette pivot per user feedback. Display swapped from Instrument Serif to **Space Grotesk** (500), body from Hanken Grotesk to **Inter**. Roman service/step numerals (i., ii., iii.) replaced with large coral display numerals (01, 02, 03). Display sizes reduced ~15% to accommodate the heavier sans grotesque.
- **v1** — 2026-05-31 — initial design pass. Editorial direction (Instrument Serif + Hanken Grotesk + JetBrains Mono on warm paper #f4ede0, logo colors as sparse accents). Full 12-section site rendered as single-file HTML with embedded CSS. Numbered section markers, asymmetric two-col layout, dark-navy launch block for rhythm. French typography fixed (nbsp before ?/:/!, non-breaking hyphen in "Parlons-en").
